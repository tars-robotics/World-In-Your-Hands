## ðŸ§° Devkit setup
Please follow these steps to make yourself familiar with the wiyh dataset:
- Read the [World In Your Hands paper]() for a detailed analysis of the dataset.
- Read the [dataset description](https://wiyh.tars-ai.com/).
- [Download](https://wiyh.tars-ai.com/exploredataset#downloads) the dataset. 
- Get the [wiyh-devkit code](https://github.com/tars-robotics/World-In-Your-Hands).
- Read the [tutorial](wiyh_tutorial.ipynb) or run it yourself using:
```
jupyter notebook ./wiyh_tutorial.ipynb
```
