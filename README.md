## 🧰 Devkit setup
Please follow these steps to make yourself familiar with the wiyh dataset:
- Read the [dataset description](https://wiyh.tars-ai.com/) for a detailed analysis of the dataset.
- [Download](https://wiyh.tars-ai.com/exploredataset#downloads) the dataset. 
- Get the [wiyh-devkit code](https://github.com/tars-robotics/World-In-Your-Hands).
- Read the [tutorial](https://wiyh.tars-ai.com/exploredataset#tutorial) or run [it](wiyh_tutorial.ipynb) yourself.
